To open the Query Interface, run the Musicology.jar file. 


The Project is implemented using Maven Architecture. The code for frontend and backend can be seen inside src/main/java/com/kde/groupe folder or can be seen by importing the this maven project into any supported Java IDE.

Following Command can be used to compile and re-run the project, if modified

Building the Project
-> mvn clean install

After building, the final jar with updated code can be found under target/swing-ui-0.0.1-SNAPSHOT.jar
